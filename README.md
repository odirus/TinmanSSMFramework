#  框架介绍
该项目为 Spring+SpringMVC+MyBatis 整合，开箱即用

# 使用方式
* 为了方便方便开箱即可用，项目下 doc 目录提供一份模拟数据供使用。导入数据库即可。
* src/main/resources 中保存项目配置文件，jdbc.properties 中存放数据库连接信息，填写正确的数据源即可。
* 运行 src/test 目录下的测试文件即可进行简单测试。
